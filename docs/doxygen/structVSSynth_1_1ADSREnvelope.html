Detailed Description

ADSR envelope data.

Represent the ADSR curve with five parameters:

Attack Amplitude Sustain Amplitude Attack Time Decay Time Release Time

An optional sixth paramater, sustainable, can also be taken. Sustainable means that the envelope can enter the sustain section of the curve. If sustainable is false, the curve will go directly from decay to release.
