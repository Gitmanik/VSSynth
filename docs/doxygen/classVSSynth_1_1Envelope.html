Detailed Description

Waveform modulator.

Modulates audio waveforms in accordance with the ADSR envelope. Envelopes can be manipulated with the hold() and release() functions.

void VSSynth::Envelope::update ( double  deltaTime)

Update the timestamp for the envelope.

While the envelope is within the Attack -> Release state, the amplitudes returned by the envelope will modulate each time this function is called. When the envelope is outside those ranges of states, this function does nothing.
